# Module 15: Finance Management (M15)

## Overview
The Finance Management module provides an end-to-end vehicle financing pipeline. It connects customers, finance executives, and administrators, allowing them to track and manage vehicle loans from initiation to final disbursement.

## Key Features
- **Application Tracking**: End-to-end tracking of finance applications from `INITIATED` to `CLOSED`.
- **Document Management**: Secure upload, validation, and storage of essential documents (Aadhaar, PAN, ITR, Form 16, Bank Statements).
- **RBAC Enforcement**: Strict visibility controls ensuring Customers see only their own applications, while Finance Executives access only their assigned applications. Admin gets full oversight.
- **Automated Notifications**: System-triggered alerts via Email, SMS, and WhatsApp hooks on critical status changes.
- **Dedicated Dashboards**: Separation of concerns between Customer-facing dashboard (for uploads and status tracking) and Admin-facing dashboard (for deep review, tracking, and updates).

## Data Models
- **FinanceApplication**: Tracks core loan details (Amount, EMI, Tenure, Bank) and state.
- **FinanceDocument**: Represents physical/digital documents, tracks their verification status, expiry date, and secure file path.
- **FinanceTimeline**: An audit trail of every status transition and explicit comment made by the agents handling the finance request.

## Endpoints
- `GET /api/finance` - List applications (with advanced filters: branch, status, executive, search)
- `GET /api/finance/:id` - View single application details
- `POST /api/finance` - Create a new application
- `PATCH /api/finance/:id` - Update application details
- `PATCH /api/finance/:id/status` - Quick status transition
- `POST /api/finance/:id/documents` - Upload required documents securely
- `GET /api/finance/:id/documents/:docId/preview` - Preview document
- `GET /api/finance/:id/documents/:docId/download` - Download document

## Status Flow
`INITIATED` -> `DOCUMENT_PENDING` -> `UNDER_REVIEW` -> `SANCTION_APPROVED` -> `DISBURSEMENT_PENDING` -> `DISBURSED` -> `CLOSED`
(or `SANCTION_REJECTED`)
