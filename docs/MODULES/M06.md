# Module M06: Vehicle Variant CMS

Extends the vehicle models by offering sub-variant specifications, pricing settings, and mechanical parameters configurations (e.g. Engine capacity, mileage).

---

# Status

✅ Complete & Production Ready

---

# Deliverables

- **APIs**: Exposes admin CRUD and public SEO endpoints (`GET /api/public/vehicles/:slug/variants`, `GET /api/public/variants/:slug`).
- **Database changes**: Mapped `Variant` with dynamic `specs` JSON column to persist extensive technical specifications and safety/comfort/exterior/interior/technology/performance features.
- **Admin pages**: Developed variant list view with pagination, search, status toggles, deletion, and duplicate features, alongside inline spec forms inside `/admin/variants/new` and `/admin/variants/edit/[id]`.
