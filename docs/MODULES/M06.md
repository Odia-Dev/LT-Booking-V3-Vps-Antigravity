# Module M06: Vehicle Variant CMS

Extends the vehicle models by offering sub-variant specifications, pricing settings, and mechanical parameters configurations (e.g. Engine capacity, mileage).

---

# Status

✅ Complete

---

# Deliverables

- **APIs**: Fetch Variants (`GET /api/vehicles/:slug/variants`), Create Variant (`POST /api/admin/variants`), Edit/Delete Variant.
- **Database changes**: Mapped `Variant` relation schema with cascade deletes constraints.
- **Frontend pages**: Spec sheets displayed dynamically under detail routes.
- **Admin pages**: Inline spec editors inside the vehicle detail panel.
