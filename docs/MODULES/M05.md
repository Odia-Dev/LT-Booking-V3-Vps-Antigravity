# Module M05: Vehicle Fleet CMS

Handles catalog listings of vehicle model lines (e.g. Fortuner, Innova). Provides creation, edit panels, status toggles, media management, and public catalog endpoints.

---

# Status

✅ Production Ready

---

# Deliverables

- **APIs**:
  - Public listings and detail queries (`GET /api/public/vehicles`, `GET /api/public/vehicles/:slug`).
  - Catalog listings (`GET /api/vehicles`), details, and administration CRUD (`POST /api/admin/vehicles`, `PUT /api/admin/vehicles/:id`, `PATCH /api/admin/vehicles/:id/status`, `DELETE /api/admin/vehicles/:id`).
- **Media Management**:
  - Supporting Hero Image, Thumbnail, Gallery Images (with ordering/removal support), Brochure PDF link, and YouTube Video URL.
- **Repository & Service**:
  - Implemented core Prisma CRUD methods, including unique slug verification, soft deletion (ARCHIVED state), sort order, price validation, and default SEO tag mappings.
- **Database changes**:
  - Extended the `Vehicle` schema in the Prisma file to host starting price, booking amount, sort order, and media reference strings.
